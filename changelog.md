# AaTempSpoof_v13.0 更新日志
> **自定义伪装控制台**
> 
## ① 新增功能
### 充电与电池管理
 * **Horae 智能控制**：新增充电时禁用 Horae，仅在充电期间自动关闭热控，充电结束后自动恢复系统原始状态。
 * **自定义电池温度墙**：支持 41°C-53°C 自定义阈值，可自动写入温度墙配置（针对欧加真机型）。
 * **全局旁路充电**：新增欧加真全局旁路充电，通过阻止电流伪装节点与充电状态控制 [测试功能]。
 * **防断记录优化**：新增欧加真模拟插拔防断记录，优化模拟插拔稳定性，降低断记录概率。
### 系统底层与性能
 * **GPU 强制渲染**：新增禁用 HW 叠加层功能，强制 GPU 直接渲染。
 * **热控管理扩展**：新增部分热控管理，增加更多热控相关处理逻辑。
 * **重触发恢复机制**：自动重触发关键配置，并恢复 105°C 温度墙。
 * **系统采样率优化**：优化全局采样率，提高系统响应速度与跟手体验，防止系统覆盖。
 * **超级省电优化**：优化移除超级省电模式，提高低电量状态稳定性并降低误触发概率。
### 适配与模式扩展
 * **米系处理器适配**：优化米系 8e 及以上处理器适配，增加对应机型电池与循环伪装节点支持。
 * **伪装白名单**：支持仅对指定应用生效伪装（该功能与充电模式互斥）。
 * **简易模式**：新增无需 WebUI 的简易模式，刷入即可直接使用。
 * **实时监控面板**：新增实时温度监控面板，支持 CPU/GPU/电池/内存温度显示，并增加实时状态指示与手动刷新。
### 安装与权限引导
 * **越狱模式适配**：安装流程新增越狱模式选择与单独适配（注：越狱模式下卸载模块需完全重启设备）。
 * **通知权限引导**：自动引导通知权限授权，优化首次使用体验。
 * **刷入流程优化**：新增权限自动配置与对应模块引导，优化整体刷入体验。
 * **模块启动通知**：模块初始化与加载完成时，将自动发送系统通知。
## ② 界面与交互升级
 * **WebUI 全新重构**：升级为多页面架构，新增状态页、配置页与插件页，视觉采用 Liquid Glass（液态玻璃）风格。
 * **动态状态页面**：新增状态卡片、运行状态动画，以及平台、设备、PID 等信息的直观显示。
 * **底部动态 TabBar**：采用液态玻璃底栏与动态指示器动画，大幅优化页面切换体验。
 * **智能隐藏逻辑**：新增充电状态限制逻辑，充电时自动隐藏部分配置，防止误操作，提高安全性。
 * **通知与公告系统**：新增公告与图片通知系统，支持模块公告弹窗与图片通知提示。
 * **WebUI 结构优化**：CSS 改为内联，JS 与 CSS 合并并增加混淆处理，同时更新了默认壁纸。
 * **必读须知提示**：WebUI 新增模块引导与伪装模块必读须知（包含风险提示与首次使用说明），首次进入模块时会自动显示。
## ③ 功能变更与优化
 * **守护循环重构**：重构启动逻辑并移除旧标记逻辑，支持多模块启动并纳入统一的守护循环。
 * **脚本交互优化**：
   * 优化 customize.sh：调整音量键检测逻辑，优化安装排版与交互体验。
   * 优化 uninstall.sh：适配新增功能，确保卸载后干净无任何残留。
 * **冲突警告机制**：新增充电加速类模块冲突警告，降低模块冲突概率。
 * **权限与兼容性**：统一模块文件权限配置，提高兼容性与稳定性。
 * **名称与描述更新**：更新模块名称与描述，完美适配当前机型显示。
 * **后台零功耗**：优化模块运行效率，后台资源占用趋近于 0%（约等于 0%），修复已知问题并提高整体稳定性。
